Conclusão

Introdução

  • Destinado a iniciantes interessados em aprender os fundamentos da programação.

  • Processing é escolhido por sua simplicidade.

  • O foco principal é ensinar como aprender a programar e criar arte interativa.

Por que Aprender a Programar

  • Programar permite desenvolver a criatividade e a autoexpressão através do potencial dos códigos.

  • Todos podem aprender a programar.

  • Os fundamentos da programação são essenciais e podem ser aplicados a outras linguagens e ambientes.

  • Trabalhar com software em um computador muitas vezes começa entendendo as limitações do software e desejando superá-las.

Visão Geral do Processing

  • Processing é um caderno de esboços flexível e uma linguagem para aprender a programar no contexto das artes visuais.

  • Simplifica muitos aspectos da programação, permitindo que iniciantes criem arte interativa rapidamente.

  • A comunidade, Processing Foundation e o apoio.

Começando com Processing

  • Baixe o instalador do Processing em processing.org para seu sistema operacional.

  • Instale e inicie o Processing.

  • Agora, analise a visão geral da interface do programa conforme vimos seus detalhes em aula.

  • Lembre-se de ir escrevendo e salvando o seu código. O processing salva os seus sketchs de projetos em arquivos PDE, um formato próprio da linguagem.

  • Caso haja alguma dúvida, não exite em consultar a documentação de referência oficial do Processing em processing.org/reference.

Conceitos Básicos de Desenho

Plano Cartesiano no Processing

  • Nas coordenadas de gráficos de computador o canto superior esquerdo é lido como (0, 0);

  • Os valores de X crescem de forma positiva para a direita

  • E os valores de Y crescem de forma positiva para baixo.

Comandos de exemplo:


size(width, height); // define o tamanho da tela.

frameRate(60); // configura a quantidade de frames por segundo.
noCursor(); // Para não renderizar o cursor do mouse na tela.

noStroke(); // Para não desenhar as linhas e contornos.
strokeWeight(10); // Para a largura das linhas e contornos.
point(x, y); // desenha um ponto.
line(x1, y1, x2, y2); // desenha uma linha entre dois pontos.

ellipseMode(CENTER); // Orienta como desenhar ellipses e círculos.
rectMode(CORNERS); // orienta como desenhar retângulos e quadrados.

Formas


rect(x, y, width, height); // para retângulos.
square(x, y, squareSize); // para quadrados
quad(x1, y1, x2, y2, x3, y3, x4, y4); // para poligonos de lados diferentes
ellipse(x, y, width, height); // para elipses.
circle(x, y, circleSize); // para círculos.
triangle(x1, y1, x2, y2, x3, y3); // para triângulos

// Diversas formas podem ser desenhadas usando seus respectivos comandos.

Cores

Em programação, os valores dados para cada cor do RGB, Vermelho, Verde e Azul, vão de '0' até '255'. Esses valores compreendem 256 Bits, o que equivale a quantidade de 32 Bytes.


// use valores de '0' até '255'

// para definir cores.
background();
stroke();
fill();

// RGB e escala de cinza: 
fill(r, g, b);
fill(g);

// Transparência com valores alfa.
fill(r, g, b, a);
fill(g, a);

Texto

// para criar texto e posicioná-lo na tela.
textSize(tamanho);
textAlign(RIGHT);
text(texto, x, y);

Você ainda pode trocar de fonte, o arquivo da fonte desejada deve ser criado pela aba de Ferramentas do procesing em "Criar Fontes...", e estar localizada em uma pasta de nome data no mesmo local do projeto: 

PFont font;
// a fonte deve estar localizada na pasta "data".
font = loadFont("LetterGothicStd.otf");
textFont(font, 128);
text("texto 1", 48, 340);

PFont mono;
// a fonte "andalemo.ttf" deve estar localizada na pasta "data".
mono = createFont("andalemo.ttf", 128);
background(0);
textFont(mono);
text("texto 2", 48, 140);

Use textWidth para saber o tamanho horizontal do texto desejado:

textSize(112);
// uso da função textWidth() para descobrir a largura da variável 'c'
char c = 'T';
float cw = textWidth(c);
text(c, 0, 160);
line(cw, 0, cw, 200); 
// uso da função textWidth() para descobrir a largura da variável 's'
String s = "Tokyo";
float sw = textWidth(s);
text(s, 0, 340);
line(sw, 200, sw, 400);

Estruturas de controle de programa

Existem duas principais estruturas que coordenam como o seu programa vai ser executado. São elas as funções setup() e draw()

O código dentro da função draw() roda continuamente de cima para baixo até que o programa seja parado. O código em setup() é executado uma vez quando o programa inicia.

// todos os comandos são lidos linha por linha, de cima pra baixo, uma por vez.

void setup (){
  // aqui os seus comandos rodam uma única vez,
  // assim que o programa começa a rodar
}

void draw (){
  // aqui os seus comandos rodam em repetição,
  // de acordo com a taxa de frames por segundo
}

Trabalhando com Variáveis

Tipos de Variáveis

  • Inteiro: int

  • Ponto flutuante: float

  • Texto: String

  • Caractere: char

  • Booleano: boolean

Esses são alguns tipos de variáveis nativas da linguagem que utilizamos no Processing. Elas podem armazenar valores de dados como coordenadas, tamanhos, cores, etc.



Atribuição e Uso

  • Atribua valores usando =.

  • Exemplos de uso:

    • Movendo um círculo com x = x + 1; ou de uma forma resumida x += 1; para animar sua posição.
    • Determinando paramentros de funções rect(x, y, size, size);
int x = 0;
int y = 0;

void setup (){
  size(600, 400);
}

void draw (){
  circle(x, y, 50);
  x += 1;
  y += 1;
}

Escopo de Variáveis

  • Variáveis Globais vs. Locais

As variáveis seguem um sistema de escopo, ou seja, elas podem ter seu valor conhecido globalbente ou localmente.

Se declararmos variáveis dentro de funções, elas vão ter seus valores conhecidos apenas por aquela função, de forma local.

Agora, se fizermos a declaração de uma variável fora do escopo de alguma função, seu valor vai ser global, e poderá ser acessado de qualquer lugar, ou função.



Interatividade

Variáveis Internas

Processing possui algumas variáveis já criadas em seu sistema, e nós podemos acessá-las para nos utilizar-mos de seus valores ao criar interações em nossos projetos.

Já conhecemos algumas delas, e aqui vão alguns exemplos de variáveis que já vimos e que iremos ver ao longo do curso:

  • mouseX e mouseY para posição do mouse.

  • pmouseX e pmouseY para posição do mouse um frame anterior.

  • width e height para dimensões da tela.

  • mousePressed para verificar o estado do botão do mouse.

  • keyPressed para verificar o estado do teclado.

Interação e Aleatoriedade

Função Random

  • Use random(max) e random(min, max) para gerar valores aleatórios.

Função de Seno e Cosseno

  • Use sin(angulo) e cos(angulo) para gerar valores oscilantes de -1 até 1.


  • Valores aleatórios e oscilantes geram valores do tipo float, e podem ser atribuídos a posições, tamanhos, cores, etc.

Condições

Estruturas de Controle

  • Declarações if e else são usados para criação de lógica de condições. Onde verificamos se tal condição é verdadeira ou falsa, e executamos comandos de acordo com o seu resultado.
  • Podemos usar variáveis booleanas ou operações matemáticas como condição das estruturas if 
int x = 50;
int y = 0;

if (x > y){
  // Aqui vai o código que será executado caso a condição do 'if' seja verdadeira.
    x = x - 5;
} else {
    // Aqui vai o código que será executado caso a condição do 'if' seja falsa.
    x = 50;
}

Laços de Repetição

  • Para iteração em variáveis e repetições usamos laços for e while.
  • Todo laço de repetição é composto por uma declaração de variável, uma condição para que ele aconteça, e por fim, uma incrementação na variável usada. 
// Em Java, existem duas principais formas de iterar com laços de repetição.
    
// Com while loops:

int i = 0;
while (i < 320) {
    line(120, i, 320, i);
    i = i + 20;
}

// E com for loops:

for (int j = 0; j < 320; j = j+20) {
    line(120, j, 320, j);
}

Podemos ter loops dentro de outros loops, chamados nested loops.
Eles são usados para iterar em mais de uma dimensão, por exemplo: 

// Para cada valor da variável 'i' passamos por todos os valores da variável 'j'.

strokeWeight(10);

for (int i = 120; i < 320; i = i+20) {
  for (int j = 0; j < 320; j = j+20) {
    point(i, j);
  }
}

Exercício 0

Rastro de Desenho

Primeiro, vamos configurar as estruturas de controle setup e draw do nosso programa.

Código Exemplo:


void setup (){

}

void draw (){

}

Agora, dentro da função setup, vamos configurar o tamanho da nossa janela com a função size(). Vamos escolher um tamanho inicial padrão, onde a largura será igual a altura da janela.

Código Exemplo:


void setup (){
  size(600, 600);
}

void draw (){

}

Escolha uma das formas geométricas aprendidas em aula, e utilize sua função respectiva para desenhá-la no centro da tela, e com o valor de um décimo da largura da tela nas dimensões de largura e altura.

O comando para desenhar a forma escolhida deve estar dentro de draw.

Código Exemplo:


void setup (){
  size(600, 600);
}

void draw (){
  ellipse(width/2, height/2, width/10, width/10);
}

Agora vamos experimentar um pouco com o que já aprendemos.

Altere os parametros de posição da forma escolhida, para que ela esteja sempre posicionada na mesma posição do mouse

Código Exemplo:


void setup (){
  size(600, 600);
}

void draw (){
  ellipse(mouseX, mouseY, width/10, width/10);
}

Vamos continuar experimentando. Conforme podemos perceber, quando não renderizamos um plano de fundo, a cada frame é desenhada a forma escolhida, causando o efeito de traçado, possibilitando a criação do rastro do mouse por onde a forma passou.

Continuando com o exercício, escolha uma cor para o preenchimento da forma utilizando a função fill(), e retire o contorno dela com a função noStroke().

Código Exemplo:


void setup (){
  size(600, 600);
  noStroke();
}

void draw (){
  fill(10, 24, 200);
  ellipse(mouseX, mouseY, width/10, width/10);
}

Crie uma nova forma geométrica, identica à forma já criada. Mantenha as duas formas sendo desenhadas, porém, na nova forma vamos inverter os parametros de posição x e y. Assim, o parametro x da primeira forma será igual ao parametro y da segunda forma, e o parametro y da primeira forma será igual ao parametro x da segunda forma

Código Exemplo:


void setup (){
  size(600, 600);
  noStroke();
}

void draw (){
  fill(10, 24, 200);
  ellipse(mouseX, mouseY, width/10, width/10);
  ellipse(mouseY, mouseX, width/10, width/10);
}

Podemos perceber a criação de um padrão espelhado na posição das formas. Mas de acordo com a movimentação do mouse, logo a tela fica completamente colorida pelas formas e não percebemos mais seu traçado.

Para resolver isso, vamos criar um fundo com uma opacidade levemente transparente. Já vimos em aula que a função padrão para gerar fundos é a background(), porém, não podemos utilizar transparência com ela.

Para resolvermos essa tarefa, vamos utilizar a função de forma geométrica rect(), criando um retângulo que tenha o tamanho da tela inteira, em sua largura e altura. E para criarmos a sua transparência, vamos utilizar a função fill() com os valores de cor e alpha.

Código Exemplo:


void setup (){
  size(600, 600);
  noStroke();
}

void draw (){
  fill(10, 24, 200);
  ellipse(mouseX, mouseY, width/10, width/10);
  ellipse(mouseY, mouseX, width/10, width/10);

  fill(0, 10);
  rect(0, 0, width, height);
}

Agora experimente livremente, por exemplo alterando os parametros do código criado ou adicionando mais formas ou cores.

A melhor forma de aprendermos é testando nossos conhecimentos e ideias.
