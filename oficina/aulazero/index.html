Introdução

  • Destinado a iniciantes interessados em aprender os fundamentos da programação.

  • Processing é escolhido por sua simplicidade.

  • O foco principal é ensinar como aprender a programar e criar arte interativa.

  • Todos podem aprender a programar.

Visão Geral do Processing

  • Processing é um caderno de esboços flexível e uma linguagem para aprender a programar no contexto das artes visuais.

Conceitos Básicos de Desenho

Plano Cartesiano no Processing

  • Nas coordenadas de gráficos de computador o canto superior esquerdo é lido como (0, 0);

  • Os valores de X crescem de forma positiva para a direita

  • E os valores de Y crescem de forma positiva para baixo.

Comandos de exemplo:


size(width, height); // define o tamanho da tela.

frameRate(60); // configura a quantidade de frames por segundo.
noCursor(); // Para não renderizar o cursor do mouse na tela.

noStroke(); // Para não desenhar as linhas e contornos.
strokeWeight(10); // Para a largura das linhas e contornos.
point(x, y); // desenha um ponto.
line(x1, y1, x2, y2); // desenha uma linha entre dois pontos.

ellipseMode(CENTER); // Orienta como desenhar ellipses e círculos.
rectMode(CORNERS); // orienta como desenhar retângulos e quadrados.

quad(x1, y1, x2, y2, x3, y3, x4, y4); // para poligonos de lados diferentes
ellipse(x, y, width, height); // para elipses.

triangle(x1, y1, x2, y2, x3, y3); // para triângulos

Escopo de Variáveis

  • Variáveis Globais vs. Locais

As variáveis seguem um sistema de escopo, ou seja, elas podem ter seu valor conhecido globalbente ou localmente.

Se declararmos variáveis dentro de funções, elas vão ter seus valores conhecidos apenas por aquela função, de forma local.

Agora, se fizermos a declaração de uma variável fora do escopo de alguma função, seu valor vai ser global, e poderá ser acessado de qualquer lugar, ou função.

Laços de Repetição

  • Para iteração em variáveis e repetições usamos laços for e while.
  • Todo laço de repetição é composto por uma declaração de variável, uma condição para que ele aconteça, e por fim, uma incrementação na variável usada. 
// Em Java, existem duas principais formas de iterar com laços de repetição.
    
// Com while loops:

int i = 0;
while (i < 320) {
    line(120, i, 320, i);
    i = i + 20;
}

// E com for loops:

for (int j = 0; j < 320; j = j+20) {
    line(120, j, 320, j);
}

Podemos ter loops dentro de outros loops, chamados nested loops.
Eles são usados para iterar em mais de uma dimensão, por exemplo: 

// Para cada valor da variável 'i' passamos por todos os valores da variável 'j'.

strokeWeight(10);

for (int i = 120; i < 320; i = i+20) {
  for (int j = 0; j < 320; j = j+20) {
    point(i, j);
  }
}

Arrays

Arrays são listas de dados, usamos para guardar elementos e valores de forma indexada em uma única variável. Os seus indexes começam a contar a partir de '0' seguindo a ordem numérica '0, 1, 2, 3,...' e assim por diante, de acordo com o tamanho da array.

  • Criar e usar arrays para armazenar múltiplos pontos de dados.
int[] numbers = new int[3]; // declara a variável 'numbers' contendo três elementos

numbers[0] = 90;  // atribui ao primeiro elemento
numbers[1] = 150; // atribui ao segundo elemento
numbers[2] = 30;  // atribui ao terceiro elemento
    
int a = numbers[0] + numbers[1]; // atribui à variável 'a' o valor de 240
int b = numbers[1] + numbers[2]; // atribui à variável 'b' o valor de 180

Ou ainda de uma forma alternativa, você pode declarar os valores de cada elemento da sua lista

int[] numbers = { 90, 150, 30 };  // syntaxe alternativa
int a = numbers[0] + numbers[1];  // atribui à variável 'a' o valor de 240
int b = numbers[1] + numbers[2];  // atribui à variável 'b' o valor de 180

    Combinando Arrays com loops

  • Também podemos iterar sobre arrays com laços de repetição.
int degrees = 360;
float[] valores = new float[degrees];
// Use um for() loop rapidamente iterar os valores de uma array.
for (int i=0; i < degrees; i++) {
  valores[i] = cos(TWO_PI/degrees * i);
}

Programação Orientada a Objetos - OOP

Classes e Objetos

  • Encapsulamento: Defina dados e métodos dentro de classes.

  • Objetos são instâncias de classes.

  • Exemplo: 

    class Ball {

    int x, y, w, h, Xvel, Yvel;

    Ball(int ballX, int ballY, int BallSize, int speed){
        x = ballX;
        y = ballY;
        w = ballSize;
        h = ballSize;
        Yvel = speed;
        Xvel = int(speed * random(0, 1));
    }

    void display(){
        fill(239, 184, 212);
        ellipse(x, y, w, h);
    }

    void move(){
        x = x + Xvel;
        y = y + Yvel;
    }

    void collide(){
        if(x > width || x < 0){
        Xvel *= -1;
        }
        if(y > height || y < 0){
        Yvel *= -1;
        }
    }
}

Construtor e Métodos

  • Uso de new para criar objetos.
Ball ball = new Ball(x, y, tamanho, velocidade);
  • Ainda podemos criar arrays de múltiplos objetos.
Ball[] balls = new Ball[5];

for (int i = 0; i < balls.length; i++) {
    balls[i] = new Ball(x, y, tamanho, velocidade);
}

void setup (){

}

void draw (){

Primeiro, vamos exercitar o nosso conhecimento de loops. Dentro da função draw, vamos criar um looping usando a função for().

Para isso, vamos recordar a sintaxe da função. Seus argumentos são compostos por uma declaração de variável, nesse caso criaremos uma variavel de nome 'i' com valor inicial de zero int i = 0. Uma condição para que ele aconteça, usaremos a condição de 'i' sendo menor do que a largura de nossa tela i < width. E por fim, uma incrementação na variável usada, para o exercício vamos somar '10' à variável 'i' i += 10.

Código Exemplo:


void setup (){
  size(600, 400);
}

void draw (){
  for(int i = 0; i < width; i += 10){

  }
}

Dentro da função for, usando a função rect() crie um retângulo que tem sua origem no canto superior esquerdo da janela, e sua largura seja metade da largura da tela, e a altura seja metade da altura da tela. Escolha também uma cor para seu preenchimento com a função fill(). E por fim, vamos usar a variável i, criada dentro da função for, para somar ao valor x de nosso retângulo.

Código Exemplo:


void setup (){
  size(600, 400);
}

void draw (){
  for(int i = 0; i < width; i += 10){
    fill(0, 12, 240);
    rect(0 + i, 0, width/2, height/2);
  }
}

Agora, vamos criar outro loop com a função for, envolvendo o loop que já criamos, e os elementos dentro dele. Dessa vez, o novo loop criado terá uma variável 'j', e sua condição será de 'j' sendo menor do que a altura de nossa tela j < height.
E então, vamos somar '10' à variável, em sua incrementação j += 10.

Agora que configuramos nosso novo loop, vamos usar a variável j para somar ao valor y de nosso retângulo.

Código Exemplo:


void setup (){
  size(600, 400);
}

void draw (){
  for(int j = 0; j < height; j += 10){
    for(int i = 0; i < width; i += 10){
      fill(0, 12, 240);
      rect(0 + i, 0 + j, width/2, height/2);
    }
  }
}

Exercício 2

Tile Map

Primeiro, vamos configurar o tamanho da nossa janela com a função size(). Vamos escolher um tamanho inicial padrão, de valor igual para a largura e a altura da janela.

Código Exemplo:

size(400, 400);

Agora, crie uma linha com a função line() que vá do canto superior esquerdo da tela até o canto inferior direito.

Código Exemplo:

size(400, 400);

line(0, 0, width, height);

Após isso, crie uma variável booleanae e atribua a ela o valor: random(1) >= 0.5.

Código Exemplo:

boolean randomTrueOrFalse = random(1) >= 0.5;

size(400, 400);

line(0, 0, width, height);

Assim, criamos uma condição, onde o valor do número aleatório gerado, que pode variar de '0.0' até '1.0', é testado contra o valor '0.5', de maneira que nossa variável booleana será verdadeira true ou falsa false, dependendo do valor aleatório.

Desta forma, podemos usar o valor em uma estrutura condicional if.

Então, vamos criar com a função if uma condição que tem como argumento a variável booleana criada.

E vamos colocar nossa linha criada com a função line(), dentro do bloco de condição if, para ela ser executada apenas se a condição for verdadeira.

Código Exemplo:

boolean randomTrueOrFalse = random(1) >= 0.5;

size(400, 400);

if (randomTrueOrFalse){
  line(0, 0, width, height);
}

Agora, crie uma estrutura else para caso nossa função if seja falsa, e dentro dela crie uma linha com a função line() que vá do canto superior direito da tela até o canto inferior esquerdo.

Código Exemplo:

boolean randomTrueOrFalse = random(1) >= 0.5;

size(400, 400);

if (randomTrueOrFalse){
  line(0, 0, width, height);
} else {
  line(width, 0, 0, height);
}

Com a estrutura que criamos, dependendo do valor da variável booleana, quando rodamos o programa visualizamos uma ou outra linha.


Agora crie uma função, e coloque dentro dela a variável booleana, e as estruturas if e else que criamos. Deixando de fora apenas a função size(), assim, ela fica fora do escopo local da função.

Dê o nome que desejar para essa função, mas lembre-se de que é aconselhado nas boas práticas da programação, que o nome de cada função deve corresponder ao que ela executa.

Os parâmetros dessa função devem ser três números inteiros, 'x', 'y' e 'tamanho', ou seja, respectivamente:, int x, int y e int tamanho. Agora, vamos usá-los para alterar a forma como as linhas são posicionadas.

Altere os parâmetros das linhas criadas dentro das estruturas if e else com os valores dos parâmetros customizáveis da nossa função.

Na primeira linha, que se visualiza apenas quando a condição if é verdadeira, vamos usar o x e o y como os dois primeiros parâmetros, e para os dois parâmetros seguintes, vamos usar x + tamanho como o terceiro parâmetro, e y + tamanho como o quarto.

A primeira função line() ficará assim: line(x, y, x + tamanho, y + tamanho);



Na segunda linha, que se visualiza apenas quando a condição if é falsa, vamos usar o x + tamanho como o primeiro parâmetro, o y como o segundo parâmetro, o x como o terceiro parâmetro, e y + tamanho como o quarto.

A segunda função line() ficará assim: line(x + tamanho, y, x, y + tamanho);



Para finalizar, crie uma função setup, e dentro dela, vamos executar a nossa função criada, logo abaixo da função size(), com os valores '0', '0' e width para seus parâmetros.

Código Exemplo:


void setup (){
  size(400, 400);
  desenhaLinha(0, 0, width);
}

void desenhaLinha(int x, int y, int tamanho) {
    boolean randomTrueOrFalse = random(1) >= 0.5;

  if (randomTrueOrFalse){
    line(x, y, x + tamanho, y + tamanho);
  } else {
    line(x + tamanho, y, x, y + tamanho);
  }
}

Perceba, que após criarmos e executarmos nossa função, obtemos o mesmo resultado onde cada linha é renderizada de acordo com o valor de nossa variável booleana.

Agora, podemos executar essa mesma função criada, porém agora dentro de um nested loop, ou seja, um loop for dentro de outro loop for, para assim, incrementarmos os valores de x e y.

Para isso, antes da função setup, vamos criar uma variável de número inteiro, uma int, para incrementar os nossos loops for, e dentro da função setup logo após a função size(), vamos declarar o valor dessa nova variável como metade da largura de nossa tela.

Vamos fazer o primeiro loop for. Como seus parâmetros vamos declarar e inicializar uma variável de número inteiro de nome 'x' com o valor '0' int x = 0. Como sua condição, vamos usar x sendo menor que a largura da tela width, ou seja, x < width. E por fim, vamos incrementar o valor de x com a variável criada fora da função setup.

Agora, para o seugndo loop for, que colocaremos dentro do primeiro, vamos declarar e inicializar uma variável de número inteiro de nome 'y' com o valor '0' int y = 0. Como sua condição, vamos usar y sendo menor que a altura da tela height, ou seja, y < height. E por fim, vamos incrementar o valor de y também com a variável criada fora da função setup.



Para finalizar, vamos mudar os parâmetros de nossa função criada para desenhar as linhas, onde o primeiro parâmetro será x, correspondendo ao primeiro loop for, o segundo será y, correspondendo ao segundo loop for, e o terceiro, o parâmetro de tamanho, será a nossa variável de incremento, criada fora da função setup, e inicializada dentro dela.


Código Exemplo:


int incremento;

void setup (){
  size(400, 400);
  incremento = width/2;
  for(int x = 0; x < width; x += incremento){
    for(int y = 0; y < height; y += incremento){
      desenhaLinha(x, y, incremento);
    }
  }
}

void desenhaLinha(int x, int y, int tamanho) {
  boolean randomTrueOrFalse = random(1) >= 0.5;

  if (randomTrueOrFalse){
    line(x, y, x + tamanho, y + tamanho);
  } else {
    line(x + tamanho, y, x, y + tamanho);
  }
}

Agora, experimente mudar o valor de nossa variável incremento para mais e menos que o valor atual, e verá como o padrão que se forma de acordo com essa alteração.

Experimente também utilizar outras formas diferentes dentro de cada condição if e else no lugar das linhas. Use sua criatividade agora.
